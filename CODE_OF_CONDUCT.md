# Code of Conduct

## Our Commitment

We are committed to providing a welcoming and inclusive environment for all contributors and community members. This Code of Conduct outlines our expectations for participation in this project and the consequences for unacceptable behavior.

## Expected Behavior

Contributors are expected to:

- Use welcoming and inclusive language
- Be respectful of differing opinions, viewpoints, and experiences
- Accept constructive criticism gracefully
- Focus on what is best for the community
- Show empathy towards other community members
- Help create and maintain a safe, inclusive space

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment, bullying, or intimidation of any kind
- Discriminatory language or jokes based on protected characteristics
- Personal attacks or insulting comments
- Trolling or deliberate disruption of discussions
- Publishing private information without consent
- Unwelcome sexual advances or attention
- Any form of violence or threats

## Reporting and Enforcement

If you experience or witness unacceptable behavior:

1. Report it to the project maintainers via email or private message
2. Include specific details and context if possible
3. Report promptly but don't feel pressured

The project team will:

- Investigate reports promptly and fairly
- Maintain confidentiality of reporters
- Take appropriate corrective action
- Communicate outcomes transparently

## Consequences

Unacceptable behavior will result in:

- Warning and request to cease behavior
- Temporary or permanent ban from the project
- Removal of contributions if necessary

## Scope

This Code of Conduct applies to:

- GitHub issues, pull requests, and discussions
- Project documentation and comments
- Community spaces and events
- Representing the project publicly
